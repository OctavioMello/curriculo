# 📄 Currículo em HTML

Este projeto consiste na construção de um currículo pessoal utilizando HTML e CSS, com foco em organização visual, estrutura semântica e apresentação profissional.

## 🚀 Objetivo
Desenvolver uma página web simples e elegante que represente minhas informações acadêmicas, habilidades e projetos.

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3

## 📌 Funcionalidades
- Estrutura semântica organizada
- Layout centralizado e responsivo básico
- Estilização moderna com CSS
- Seções de formação, habilidades e projetos
